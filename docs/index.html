Zoph

Zoph (Zoph Organizes PHotos) is a web based photo management system written using PHP and backed by MySQL. Photos are loaded to the db using a perl script or a web interface. Zoph can be found on http://www.zoph.org

At first, I will simply copy the current documentation into Wiki and do a rough edit of the layout. When all the current documentation is available in this Wiki, I will do a finer edit of all the pages, to make it into a proper user manual.

This document will grow to be the main documentation source for Zoph users and administrators.
